Explanation

Ranking
(Least Specific to Most Specific) 		Specification Type Explanation
3 Type Specific Targets ALL the elements of a given HTML tag type

(ie. specifying <td> as a CSS celector will target ALL <td> elements unless overriden)
2 Class specific Targets ONLY the elements with the specific 'class' attribute, regardless of type specification

(ie. I gave the class attribute 'class-specific' to the 2nd and 3rd data cells in the test column and styled them differently than the element specific styling)
1 Id specific Targets ONLY the elements with the specific 'id' attribute, regardless of class or type specification

(ie. I gave the id attribute 'id-specific' to the 3rd data cell in the test column and styled it differently than the class specific styling)

Conlusion

'Id' specific selectors have the most authority!

'Class' specific selectors have the 2nd most authority. If an 'id' specific selector is not present, the a class specific selector will have the authority!

'Type' specific selectors have the least authority. If no other selectors are present other than the type of the tag itself (ie. <p>), then the type selector has no other choice but to have authority!
